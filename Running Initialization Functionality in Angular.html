The environment injector is a generalized version of NgModuleRef, which is also known as the ‚Äúmodule

In my application, I‚Äôve created a function named attachInitEffect, which takes an init function and registers it in the ENVIRONMENT_INITIALIZER token:


import { ENVIRONMENT_INITIALIZER, Provider } from '@angular/core';

export function attachInitEffect(fn: VoidFunction): Provider {
  return {
    multi: true,
    provide: ENVIRONMENT_INITIALIZER,
    useValue: fn,
  };
}

      duration: 10_000,

  })
